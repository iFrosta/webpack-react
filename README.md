# Webpack-React template
template for quick start of project development

Build dev version ( without minify, hash, etc)
```npm
npm run dev 
```
Build dev version and run file watcher ( without minify, hash, etc)
```npm
npm run watch
```
Run server on port: 4200
```npm
npm run start
```
Build project
```npm
npm run build
```
Stats
```npm
npm run stats
```
